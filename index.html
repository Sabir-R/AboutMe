The Process

First off, I went to the Start Bootstrap website to find a template to use for this website. Going through the templates, I chose the Grayscale template they had available because I liked the dark theme they had with the template.

Decisions Made

The first decision I had to make was what to replace the images with. I decided to stick with the dark theme so I found pictures that fit with the theme. Then I had to customize the template with my own font colors.

Modifications Made

I changed the color for the fonts. I also changed the pictures so they still fit in with the dark theme. I also changed the headings for the navigation bar so it fit my needs.

What was Easy?

It's simple to change most of the text from default to custom. it was also easy to add my own classes in the css files so I had my own custom colors.

What was Hard?

It took me a while to figure out where to go in the files to change the templates headings on the navigation bar. It also took me a while to replace the pictures from the template with my own.
